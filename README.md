## 1. 라이센스: CC BY-NC-SA
  - 저작자 표시 (Attribution, BY): 자료를 사용하는 사람은 반드시 원작자를 표시해야 합니다.
  - 비영리 (NonCommercial, NC): 상업적 용도로 사용할 수 없습니다.
  - 동일조건 변경허락 (ShareAlike, SA): 자료를 수정하거나 2차 저작물을 만들 경우, 원작과 동일한 라이선스(즉, CC BY-NC-SA)로 다시 공유해야 합니다.
